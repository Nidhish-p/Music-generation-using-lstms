Music Generation using LSTM

(Long Short Term Memory)

About Our Project

Our project, Music Melody Generator using LSTM, aims to create a deep learning model capable of generating musical sequences. Using a dataset of monophonic music, we preprocess the data by converting it into MIDI and kern formats, ensuring consistency through transposition and note duration filtering.

We then encode the music into a time-series format and train an LSTM model to learn musical patterns and generate melodies. LSTMs are particularly suited for this task as they effectively capture long-term dependencies in sequential data, making the generated music more coherent and structured.

Here are the steps involved in our project:

Step 1: Data Collection

We gathered a diverse dataset of monophonic music pieces, ensuring high-quality musical patterns for training the LSTM model to generate coherent melodies.

Step 2: Data Conversion

The collected MP3 files were converted to MIDI format for structured representation, then further extracted into kern format for efficient analysis and preprocessing, ensuring consistency in musical patterns.

Step 3: Data Preprocessing

We refined the dataset by filtering acceptable note durations, transposing songs to a common key (C major or A minor), and encoding musical data into a structured time-series format for LSTM training.

Step 4: Data Maping

We assigned unique integer values to musical notes and stored them in a JSON mapping file, converting encoded songs into numerical data for efficient LSTM training.

Step 5: Model Traning

The LSTM model was trained on encoded musical sequences, adjusting weights iteratively to minimize prediction error and learn melodic patterns effectively.

Step 6: Music Generation

Using a trained LSTM model, new melodies are generated by predicting the next notes in a sequence, ensuring coherence and musicality.
